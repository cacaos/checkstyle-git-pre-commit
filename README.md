#### 路
**javaEE @PostConstruct**
<br/>
被 `@PostConstruct` 修饰的方法会在服务器加载Servlet的时候运行，并且只会被服务器执行一次
<br/>
<br/>
**git hooks**
<br/>
pre-commit 钩子在键入提交信息前运行。 它用于检查即将提交的快照
<br/>
<br/>
**Checkstyle**
<br/>
Checkstyle是一种开发工具，用于帮助程序员编写符合编码标准的Java代码。
